# 我是杨





## 前言

*讲一讲为什么建这个仓库*

* 直接目的：向hr展示我是一个怎么样的人；**深层诉求**：向世界展示我是一个什么样的人。建这个仓库是一种**强烈的冲动**，我在寻求一种身份认同与世界连接的方式。
* 我想过这个仓库可能会带给我麻烦，因为我暴露了自己的缺陷与不足。但是只要我在hr审视我之前，达到工作基本要求然后展现我的潜力不就行了吗。*（**我应当有有这个自信**。在2026.06.09这一天我确立了自己要做一个”境界派“（来自《夜无疆》），然后把”事功“（来自《剑来》）作为准则。我要先拿到”最少且必要的知识“，提升境界，再边做边补，打磨境界。重实效，结果导向。）*
* 一些其他作用：**作为督促，促进我的进步**，**把生活中零散的灵感记录下来，方便整合成一套流程，形成自己的生活哲学**。



## 我的渴求

*包含我最强烈的想法*

**关注自我，连接世界**。然后成长为一个很酷的人。

* 过去我很多时候忽略了自己的感受，或者只关注到了情绪。我想更深层地关注自己，包括身体层面，延伸自己的**感知**，确认**自己存在**的事实，享受当下。
* 我有想向外界表现自己的诉求。我想让别人觉得我很厉害，有思想和内涵，还很帅。我会一步步打磨自己。现在的我在学习，在健身。
* 一个更大的想法，影响外界。没有说”改变“和”世界“这两个词，是因为它太大了，有点脱离实际情况。我的每一步都要坐实，由内向外，我的目标要切实可行，每一个节点都有轨迹。





## 我现在的选择与未来可能

*我选择计算机行业，现在打算做游戏开发，未来有可能更改的方向*

* 我选择计算机行业，是把计算机当作自己开拓世界的武器。我将用它展现***力量和勇气***。
* 现在打算做游戏开发，首先是想借此**表达自我**。我认为做游戏就是创作，和写书、做动漫都是一个性质。我想在游戏开发里展现我的审美、我想传达的理念、想讲好的故事。*那你为什么不往编剧和美术方面发展呢*？这是我想说的第二点，也许我是想**让我认同的变得更好，让更多人认同和喜欢**。别人的美术作品和文学作品，只要被我喜欢，我就想让其他人也喜欢，不一定要我自己实际塑造角色形象，去画每一幅画。因为我喜欢代表了我认同作品的美，它表达的理念和我想的一样，这何尝不是展现自我呢？然后就是**不满足现状**。现在游戏还不能呈现一个完美的虚拟世界，角色会穿模，低配置下的展现效果不尽人意等等。有时候画师画的立绘很符合我的审美，但建模却一点也不灵动，服装质感也不行，让我不满意。我要加入游戏开发行业，去创造更真实、更美好的游戏世界。我不会等待英雄来解决问题，而是**做自己的英雄**（《来自冰海战记》）。**我想加入游戏开发的前沿，加入到大家的讨论里，让我们喜爱的游戏更好**。
* 未来方向：我现在是打算做游戏客户端开发`Gameplay程序`，然后打算向`图形程序`发展。再就是受现在AI的发展影响，将来可能会研究`使用AI解决游戏问题`，`算法`等，如游戏资产轻量化算法研究等。但是我现在还不知道其中的职能分化和细节差异等等，不知道其中水有多深，这**之后的路留待以后选择**。但是我的**愿景**是不断深耕领域，努力走到游戏开发的前沿。



## 我的成长历程

*包含实际经历和心路历程*

* 2026.06.10，现在我22岁，我一无所有。那么5年后呢？等我到30岁的时候呢？
* 我是一个`看起来很闷`的人，总是一个人呆着：看动漫、小说，玩游戏，出去散心，偶尔在需要的时候独自学习。其实我是个`情感很丰富`的人，内心戏十足，不过我习惯把自己的情感藏在内心，在虚拟世界寻求情感寄托与共鸣。不过那是之前，现在的我`想要展现自我`，渴望被人发现认同；可能是看多了主角们拯救世界、创造奇迹的故事，我有些`理想主义甚至好高骛远`。之前我并不清楚我想要做什么，但是我总有一种期望：那就是做到非常好。比如学习，初中想要进入全市最好的高中，高中想要进入全国top前列的有名学府。但是我的期望并不能化为动力，相反我还很厌恶学习，只完成基本学习内容，没有大目标只是想要好成绩，如果有外部压力，就会强迫自己去学，我的内心总是在逃避学习，这使得我看起来挺认真的但一事无成。现在的我，仍是`理想的`，我坚信”世上无难事，只要肯攀登“，不过我会关注自己的诉求，让目标更贴合自己实际，我也是`现实的`。比如我现在喜欢有些肌肉的薄肌型，而不追求备受健身人追捧的力量，我会反复调整训练方式达到我想要的肌肉形态。这些话还有很多很多可以展开说的，专门开一个笔记吧。
* 我的经历：我将自己的经历分为幼时，儿童，年少，待成熟，以及待发展五个阶段。`幼时`喜欢嬉闹，爱跑爱跳，爱出去玩，嬉闹这一特性直到年少时仍保留。喜欢哭闹，内心少有烦躁就要大喊大叫，可见我从很小时就能察觉自己的情绪变化。`儿童`时期大概在小学，这一段时间我就开始把自己关到屋子里了，放学了大家都回家了，好无聊，于是我就开始看电视，后来是玩手机游戏。再然后班上有人看《斗罗》，我开始了看漫画。可能是我的母亲担心我一直看手机不好，她给我在2元店淘到了一本《偷星九月天》的同人小说，还有盗版的《吞噬星空》，加上漫画更新慢，我又想看到更多剧情就主动购买小说的缘故，我迷上了网文。因为光碟内容有限以及互联网的普及，我开始了线上追动漫的生活。游戏、小说、动漫填补了我现实的空虚，却也带来了一道墙——与外界交流的墙，我不敢迈出我的舒适圈。这片”无尘之地“（来自《龙族》），保护了我，也隔绝了我与现实世界。`年少`时，总是怀有远大愿景，好高骛远，擅自期待又擅自失望，没有在失败中前进而只是走过了岁月，让风带走忧愁。同时我没有主见，困在父母给自己的选项里。要好好读书，于是我就只是读书，本质上也不是学习，只是记住了知识而已。之所以不用”目标“这一词是因为我从没有找到我愿意努力实现的理想，那个愿景称不上”目标“，只是一种完美主义下的畸形产物罢了。`待成熟`时期是高中与大学前中期，我保持着对未来困惑。前期高中时十分好强，会因为想要有好成绩而选择熬夜，却又因为厌恶学习而偷懒（这是指在校期间努力，而放家偷懒，思想上也不擅长总结经验，困在舒适圈里）。这时候的我很极端。为了追求好成绩，我变得很敏感，对周围人的嬉闹与闲聊感到厌烦。我会在下课后仍然埋头看书，会对晚上熄灯后的闲聊尤其厌恶。另一方面我又在寻求解脱，晚饭不吃去打羽毛球，去跑步。这种矛盾呈现了一副奇怪现象：平时努力的我，在高考期间，竟然打羽毛球，在老师劝戒下仍不愿离开。然后我第一次高考考砸了。因为我所谓的自尊心，我选择了复读。在新的学校里，更强的班级里学习。然而我的问题始终没有解决，我与我未达成和解。上了大学，我逃课，不学习，每一天在宿舍看小说、动漫，打游戏，这使得我大一挂了3门学科。我感觉我在摆脱束缚，我在尝试做出自己的选择。尽管结果不利于后续发展的方面，但我不后悔。这期间我渐渐感觉到生活的无聊，周围人对我的轻视进一步加深了我想要改变现状的诉求。到了大三下，我察觉到父母的衰老，我想要扛起责任，开始仔细思考未来方向。`待发展`方向是从前不久开始的，转变是我关注自己身体感受，想要更好看就护肤和健身，有想法就去实现。我想要自身强大，渴望表现自我，但也不依赖他人。



## 留待完善的自我

我追求事功。想要读一些哲学思想，把思想作为武器指导行动。然后要把技术打磨好，形成一套得心应手的技术链。社交做到不卑不亢，不迎合失去自我，也不要不顾他人感受。
